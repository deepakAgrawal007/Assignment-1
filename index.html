# Illustrate the structure of an HTML table using the table, tr, th and td elements. Provide an example of a table with headers and data cells?

Answer: Structure of an HTML table:

  1. table tag is the parent element of html table
  2. tr tag is the child of table tag and is used to create table rows
  3. th tag is used to create the table heading of the tables
  4. td tag is the child of table tag and is used to create data cells

# Explain the significance of the colspan and rowspan attributes in table cells. How can these attributes be used to create complex table layouts?

Answer:

  • Colspan: it is an attribute that defines the number of columns a cell should span.
  • Rowspan: it is an attribute that defines the number of row a cell should span.
